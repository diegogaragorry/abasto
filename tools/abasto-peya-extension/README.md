# Abasto PeYa Sync

Extension local de Chrome para sincronizar PedidosYaMarket con Abasto sin pegar cookies, request ni user-agent.

## Instalacion

1. Descargar `abasto-peya-extension.zip` desde Abasto.
2. Descomprimir el archivo.
3. Abrir `chrome://extensions`.
4. Activar `Modo desarrollador`.
5. Elegir `Cargar descomprimida`.
6. Seleccionar la carpeta descomprimida `abasto-peya-extension`.

Despues de instalarla, volver a Abasto y usar `Sincronizar PeYa con extension`.
